Istruzioni

Requisiti
Python 3.6 o superiore
tkinter (pre-installato con Python) per la versione GUI
python-magic (opzionale): pip install python-magic
Come installare Python:

Per installare GRATUITAMENTE Python:

Clicca su Download Python 3.x.x (la versione più recente) dal sito ufficiale.

Molto importante: nella schermata di installazione, spunta la casella

✅ "Add Python to PATH"

prima di cliccare su "Install Now".

Una volta installato, avvia il file start.gui

Esecuzione
  • Versione Web:
    Apri il file HTML scaricato in un browser qualsiasi
  • Versione GUI:
    Estrai ZIP e fai doppio click su start_gui.bat (Windows) o start_gui.sh (Mac/Linux)
  • Versione CLI:
    Estrai ZIP e usa process_zip.bat file.zip (Windows) o ./process_zip.sh file.zip (Mac/Linux)

Funzionalità

Separazione Immagini

Estrae e raggruppa tutti i file immagine (JPG, PNG, GIF, SVG, ecc.) in un unico ZIP.

Separazione Audio

Raccoglie tutti i file audio (MP3, WAV, FLAC, ecc.) in un archivio dedicato.

Separazione Video

Organizza tutti i file video (MP4, AVI, MKV, ecc.) in un file ZIP separato.
